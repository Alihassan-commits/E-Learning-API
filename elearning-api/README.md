# 📘 E-Learning API

A RESTful API built with Laravel for an online learning platform.  
It provides authentication, course management, and a commenting system using a clean service-based architecture.

---

## 🚀 Features

- 🔐 User Authentication (Register, Login, Logout)
- 📚 Course Management (CRUD operations)
- 💬 Comment System for Courses
- 🧠 Service-based architecture (clean code separation)
- 📦 RESTful API design
- 🧾 OpenAPI / Swagger documentation support

---

## 🛠️ Tech Stack

- PHP 8+
- Laravel Framework
- MySQL
- Laravel Eloquent ORM
- OpenAPI (Swagger)
- Git & GitHub
